CToledo GPT
CToledo GPT is an AI assistant designed for networking, especially with potential employers. It provides a convenient way for anyone to learn about the creator's professional life by accessing details about experience, education, and projects.

This AI assistant is powered by GPT-4o-mini, and it allows users to ask any question, with a primary focus on answering questions about the creator's background.

Technologies Used
Frontend: React, TailwindCSS, TypeScript
Backend: Node.js
API: GPT-4o-mini for AI-based responses
How It Works
The main feature of CToledo GPT is its ability to answer any question. While users can ask anything, the AI is particularly tailored to answer questions related to the creator’s professional life, including:

Education
Work experience
Projects
Access
No installation is required. You can access the project directly by visiting the following URL:

https://ctoledogpt.onrender.com/

Once there, simply ask any question and CToledo GPT will provide an answer!

Usage
Open the website: https://ctoledogpt.onrender.com/
Ask questions (focused on professional life or anything else).
The AI assistant will respond based on the information available.
Features
Interactive AI Assistant: Ask anything about the user’s professional life.
GPT-4o-mini: Utilizes the GPT-4o-mini API for generating responses.
User-Friendly Interface: Built with React and styled using TailwindCSS for a responsive, clean UI.
